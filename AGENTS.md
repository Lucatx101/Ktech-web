# Ktech Việt Nam Website Agent Notes

- This repository is the VI/EN corporate profile website for Ktech Việt Nam.
- This is not an e-commerce website.
- Core slogan: `Understand the universe`.
- Visual direction: `Industrial Intelligence`.
- Ktech has five capability groups: enterprise AI, automation and special machinery, precision CNC, automotive support, and technical chemicals.
- Do not invent certifications, customers, partners, projects, results, or production capacity.
- Approved source assets in `assets/source-approved/` are immutable. Do not delete, move, rename, overwrite, recompress, or otherwise modify them.
- Keep source content separated from components. User-facing copy should come from dictionaries or content modules.
- Use Server Components by default.
- Prioritize accessibility, responsive behavior, performance, and technical SEO.
- Do not add dependencies without a concrete need.
